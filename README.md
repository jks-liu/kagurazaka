# kagurazaka
Kagurazaka agent is a general HTTP/HTTPS proxy. It can over the wall by default according to the hosts from [https://github.com/racaljk/hosts][].

# Usage
1. Install python
2. Install dependencies
``` $ pip install requests ```
3. Run
``` $ python kagurazaka.py ```
